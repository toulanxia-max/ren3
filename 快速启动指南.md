# 《忍者必须死3》家族网站快速启动指南

## 🚀 一键启动（推荐）

### Windows系统
```powershell
# 启动所有服务（需要管理员权限）
.\启动所有服务.bat

# 或分别启动
cd backend
npm start

# 新终端
cd frontend
node node_modules/react-scripts/bin/react-scripts.js start
```

### 创建启动脚本
创建文件 `启动所有服务.bat`：
```batch
@echo off
echo 启动《忍者必须死3》家族网站服务...
echo.

echo 1. 启动MySQL数据库（确保MySQL服务已运行）
net start mysql >nul 2>&1 || echo MySQL服务未启动，请手动启动

echo 2. 启动后端API服务...
start /B cmd /c "cd backend && node src/server.js"

echo 3. 等待后端启动（3秒）...
timeout /t 3 >nul

echo 4. 启动前端网站...
start /B cmd /c "cd frontend && node node_modules/react-scripts/bin/react-scripts.js start"

echo 5. 等待前端启动（5秒）...
timeout /t 5 >nul

echo.
echo ✅ 服务启动完成！
echo.
echo 🔗 访问链接：
echo 前端网站: http://localhost:3000
echo 后端API: http://localhost:5000
echo 健康检查: http://localhost:5000/health
echo.
pause
```


## 🔧 常见问题解决

### 问题1: 前端启动报错 "node不是内部或外部命令"
**解决方案**:
```bash
# 方法1: 直接运行react-scripts
cd frontend
node node_modules/react-scripts/bin/react-scripts.js start

# 方法2: 检查Node.js安装
where node
node --version
```

### 问题2: 后端连接数据库失败
**解决方案**:
1. 检查MySQL服务是否运行
   ```bash
   net start mysql
   ```
2. 检查数据库配置
   ```bash
   # 测试MySQL连接
   mysql -h localhost -P 3306 -u root -p111111
   ```
3. 修改后端 `.env` 文件中的数据库配置

### 问题3: 前端访问显示空白页或404
**解决方案**:
1. 检查前端是否编译成功
2. 清除浏览器缓存
3. 打开开发者工具（F12）查看控制台错误
4. 检查代理配置是否正确

### 问题4: 注册失败"验证失败"
**解决方案**:
1. 检查用户名是否包含特殊字符
2. 确认密码长度至少6位
3. 邮箱字段留空（不填）
4. 确认两次密码输入一致

## 🔄 服务管理

### 查看服务状态
```bash
# 查看进程
tasklist | findstr node

# 查看端口占用
netstat -ano | findstr :3000
netstat -ano | findstr :5000
```

### 停止服务
```bash
# 停止前端（查找进程ID并终止）
taskkill /F /PID <前端进程ID>

# 停止后端（查找进程ID并终止）
taskkill /F /PID <后端进程ID>
```

### 重启服务
```bash
# 停止所有Node进程
taskkill /F /IM node.exe

# 重新启动
cd backend && node src/server.js
cd frontend && node node_modules/react-scripts/bin/react-scripts.js start
```
